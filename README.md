# 🤖 AI Mock Interviewer

A modern AI-powered web app that generates interview questions and answers based on your selected topic. Built for students and developers to practice interviews effectively 🚀

---

## 🌐 Live Demo

👉 https://your-site-name.netlify.app

---

## ✨ Features

* 🎯 Generate interview questions by topic
* 🔢 Custom number of questions 
* ⚡ Fast AI response using Groq API
* 📦 Chunk-based generation (no cutoff issues)
* 🎨 Clean & modern UI
* 📖 Large readable question/answer cards

---

## 🛠 Tech Stack

* React.js
* CSS (Custom styling)
* Groq API (LLM - llama-3.1-8b)
* Netlify (Deployment)
* GitHub (Version Control)
